# Консольный калькулятор с историей

Простое консольное приложение на Python для выполнения базовых математических операций (сложение, вычитание, умножение, деление) с сохранением истории всех вычислений в файл `history.json`. Проект создан в учебных целях с использованием инструментов искусственного интеллекта для генерации кода, обработки ошибок и модульной структуры.

## Возможности

- Выполнение арифметических операций над двумя числами (целыми или с плавающей точкой).
- Команды: `add`, `sub`, `mul`, `div`.
- Обработка ошибок: неверный формат ввода, деление на ноль, некорректные символы.
- Сохранение истории операций в JSON-файл при выходе и автоматическая загрузка при старте.
- Просмотр истории по команде `history`.
- Удобное консольное меню с подсказками.

## Структура проекта
console-calc/

├── parser.py # разбор введённой строки на операцию и операнды
├── operations.py # выполнение арифметических операций
├── history.py # класс для работы с историей и сохранения в JSON
├── .gitignore

## Требования

- Python 3.6 или выше (используются f-строки, аннотации типов).
- Стандартная библиотека (дополнительные пакеты не требуются).

## Установка и запуск

1. Клонируйте репозиторий:

   git clone https://github.com/ваш-username/console-calc.git
   cd console-calc
2. Запустите приложение:

bash
python main.py
Использование
После запуска появится приглашение >>>. Доступные команды:

Выполнение операции:
add 2 3 → результат 5.0
sub 10.5 0.5 → результат 10.0
mul 4 5 → результат 20.0
div 9 2 → результат 4.5

Просмотр истории:
history → вывод списка всех предыдущих операций.

Выход:
exit — сохраняет историю в history.json и завершает работу.

Обработка ошибок
Неверный формат: mul 5 → Ошибка: нужно ввести ровно 3 части: операция число1 число2

Неизвестная операция: pow 2 3 → Ошибка: неизвестная операция 'pow'

Деление на ноль: div 10 0 → Ошибка: деление на ноль

Логика разбиения на модули и структура проекта также предложены ИИ.
